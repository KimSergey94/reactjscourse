1. Компонент карточки состоит из следующих составляющих в интерфейсе для мобильных устройств (по порядку сверху вниз):
    1.1 Аватар и имя автора поста, и время когда был опубликован пост.
    1.2 Кнопка меню поста
    1.3 Заголовок

    1.5 Кнопки управления

2. В интерфейсе для планшетных устройств, порядок слева на право таков:
    2.1 Контент
    2.2 Заголовок (по середине - вверху)

    2.4 Кнопки управления (статистика, поделиться, удалить) (по середине - внизу)
    2.5 Dropdown список с остальными кнопками управления
    2.6 Рейтинг
